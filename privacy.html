プライバシーポリシー

制定日: 2026年6月10日

「未読ゼロ」（以下「本サービス」）は、Gmail の未読メールを一括で既読にするためのウェブアプリケーションです。本ポリシーは、本サービスがユーザーの情報をどのように扱うかを定めるものです。

1. いちばん大切なこと

本サービスにはサーバーがなく、メールの内容・件名・送信者などのデータ、および認証トークンを、開発者側のサーバーへ送信・保存することは一切ありません。すべての処理は、ユーザーのブラウザと Google のサーバーとの間で直接行われます。

2. 取得する情報と利用目的

  • Google アカウントへのアクセス許可（Gmail の変更権限）: 未読メールの一覧取得と、未読ラベルの除去（既読化）のためだけに利用します。
  • アクセストークンはブラウザのメモリ上にのみ保持し、Cookie・ローカルストレージ等への保存は行いません。ページを閉じると消滅します。
  • メールの本文・件名・添付ファイルを読み取り、表示、保存することはありません。処理に使用するのはメールの ID と未読件数のみです。

3. Google API の限定的使用（Limited Use）について

本サービスによる Google API から取得した情報の使用は、Google API Services User Data Policy（限定的使用の要件を含む）に準拠します。取得したデータを広告目的で利用すること、第三者へ譲渡・販売することはありません。

4. アクセス解析・Cookie

本サービスは、アクセス解析ツールおよび Cookie を使用していません（Google のログイン機能が技術上使用するものを除きます）。

5. アクセス許可の取り消し

ユーザーはいつでも Google アカウントの設定 から、本サービスへのアクセス許可を取り消すことができます。

6. 改定

本ポリシーを改定する場合は、本ページにて告知します。

7. お問い合わせ

本サービスおよび本ポリシーに関するお問い合わせは、tokiyoshi.0410@gmail.com までお願いします。
